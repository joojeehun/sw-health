# sw-health
프로젝트명 : 건강 피드백
배경 및 문제 정의 : 잘못된 생활 패턴을 피드백 주려고 함.
목적 : 다양한 사용자의 생활 패턴 분석 및 피드백 
예상 사용자 : 나이 무관, 모든 사용자
기능 요약 : 사용자의 생활 패턴을 입력 받고 AI를 통해 분석하여 사용자의 연령에 맞게 식습관, 수면패턴 등 고칠 점을 피드백한다.

"본 프로젝트는 AI 건강 피드백 시스템을 개발한다. 현재 사람들은 자신의 생활 패턴이나 식습관, 필요한 영양소에 대해 알지 못 하는 상황이 대부분이다. 
본 시스템은 사용자 생활 패턴 분석, 이에 따른 고칠 점 피드백과 같은 기능을 제공하여 이 문제를 해결한다. 
사용자는 전 연령이 사용할 수 있게 할 것이며, 모바일 앱 기반으로 제공할 예정이다."
